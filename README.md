# Text Library

一个移动端优先的本地文案管理工具，适合整理追星文案、活动文案、日常灵感文案，以及各类可重复使用的文本素材。

## 项目定位

Text Library 面向两类典型场景：

- 追星文案工具：用于整理应援文案、安利文案、控评文案、活动宣传文案、图片配文等内容
- 通用文案工具：用于管理短句素材、品牌文案、社媒文案、客服快捷回复、灵感备忘等文本条目

它强调的是“本地轻量管理”，不依赖服务端数据库，打开网页即可使用，适合个人高频整理和快速调用。

## 核心功能

- 支持导入本地 Excel / CSV 文件
- 自动按行识别为独立文案条目
- 支持目录分类管理
- 支持搜索、排序、分页浏览
- 支持复制后自动归入历史记录
- 支持历史记录恢复
- 支持单条删除、目录删除
- 使用 IndexedDB 在浏览器本地持久化保存数据

## 适合的使用方式

- 为不同艺人、不同站子、不同活动建立独立目录
- 导入追星素材库，快速复制需要的文案内容
- 为日常社媒运营建立常用文案仓库
- 为个人网站作品集展示“真实可交互的效率工具型 Web App”

## 技术实现

- 原生 HTML / CSS / JavaScript
- SheetJS 解析 Excel / CSV
- IndexedDB 本地存储
- 移动端优先界面

## 数据说明

- 所有数据默认保存在用户自己的浏览器本地
- 不同设备、不同浏览器之间不会自动同步
- 清除浏览器站点数据后，本地文案内容可能被移除

## 预览与部署

- 本地开发可通过静态服务器直接运行
- 可部署到 GitHub Pages、Vercel 等静态托管平台
- 适合嵌入个人网站、作品集页面和简历项目展示
